The problem

Image analysis is fragmented.

Researchers and physicians juggle disconnected tools for viewing, segmenting, measuring, and reporting. DICOMclaw replaces that with a single AI agent.

Traditional workflow

  • Switch between viewer, segmentation tool, spreadsheets
  • Manual region drawing, copy-paste measurements
  • Separate software for each analysis step
  • Results disconnected from the images

With DICOMclaw

  • One workspace: viewer, agent, and results together
  • Describe your analysis in natural language
  • Agent plans, executes, overlays results on the viewer
  • Every result is linked to the image it came from

Key features

What the agent can do.

From viewing raw DICOM data to generating structured reports — the agent handles the entire pipeline while you stay in control.

01

Integrated Viewer

Multi-panel display with CT, PET, fused, and MIP views. Navigate, adjust, and interact — all connected to the agent's analysis.

02

Autonomous Agent

An AI that reasons about your request, proposes an analysis plan, and executes it step by step — with your approval at every stage.

03

Automated Quantification

Lesion detection, organ segmentation, uptake measurement, and radiomic feature extraction — from a single request.

04

Interactive Drawing

Draw regions of interest directly on the viewer, refine thresholds, and reference them in conversation for targeted analysis.

05

Conversational Reading

Ask the agent to run analyses, or chat with a vision model about what you see. The agent sees the same images you do.

06

Structured Reports

All findings — segmentations, measurements, interpretations — compiled into a structured report linked to original images.

How it works

Talk to your images.

Describe what you need in plain language. The agent understands clinical context, plans the analysis, and delivers results on the viewer.

"Segment the liver and measure uptake"
The agent segments the organ, computes statistics, and overlays the contour on the viewer.
"Find all lesions and generate a report"
Whole-body detection, quantification of each finding, and a structured summary.
"What do you see in this scan?"
The vision model interprets the current view and discusses findings with you.
