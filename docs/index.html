Capabilities

Everything in one workspace

Converse with an AI agent that sees what you see, operates directly on imaging data, and presents results back into the viewer in real time.

DICOM Viewer

2x2 grid with CT, PET, Fusion, and MIP viewports. W/L presets, PET colormaps, slice sync, and orientation switching. Powered by Cornerstone3D.

ReAct Agent

Plan-and-execute reasoning loop with transparent thinking, plan approval before multi-step analyses, and real-time progress updates.

Quantification

AutoPET-3 lesion detection, TotalSegmentator organ segmentation (117 structures), SUV statistics, radiomics (GLCM, shape, first-order).

Interactive

Draw VOIs on PET/Fusion, click MIP for navigation, @mention VOIs in chat, threshold refinement — direct manipulation meets conversation.

Dual Chat Modes

Agent mode for full skill execution with plan approval. Chat mode for direct VLM conversation using live viewport snapshots.

Markdown Skill Guides

Each skill is controlled by an editable markdown file injected into the agent prompt at runtime. Change behavior without touching code.

How it works

Converse, not configure

Describe what you need in natural language. The agent plans the analysis, asks for your approval, executes the skills, and overlays results directly on your viewer.

  • "Segment the liver and calculate SUV"
    Agent proposes a plan → you approve → TotalSegmentator runs → SUV stats computed → contour overlay appears on viewer
  • "Detect all lesions and generate a report"
    AutoPET-3 inference → lesion quantification → structured markdown report with MIP overlay
  • "Extract texture features for @VOI1"
    GLCM (2D/3D), shape, first-order radiomics → CSV + radar chart visualization

System Design

Architecture

Three-layer design connecting an interactive viewer with an autonomous reasoning agent.

Frontend React 18 + Vite  |  Cornerstone3D v4.18  |  Chat UI  |  VOI Drawing  |  Plan Approval
│ Socket.io + REST
Backend Node.js + Express  |  Socket.io  |  SQLite  |  WADO-URI  |  VLM Client
│ subprocess (stdin / stdout / stderr)
Agent Python ReAct Loop  |  10 Skills  |  Markdown Guides  |  Clinical Context
│
Models AutoPET-3 (nnU-Net)  |  TotalSegmentator  |  VLM  |  OpenRouter LLM
